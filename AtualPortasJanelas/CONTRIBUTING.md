# Guia de Contribuição

## Como Contribuir

Bem-vindo! Estamos felizes que você queira contribuir para o projeto Atual Portas & Janelas.

### Fluxo de Desenvolvimento

1. **Fork** o repositório
2. **Clone** seu fork localmente
3. **Crie uma branch** com uma nomenclatura clara
4. **Desenvolva** sua feature/correção
5. **Teste** suas mudanças
6. **Commit** com mensagens descritivas
7. **Push** para seu fork
8. **Abra um Pull Request**

### Nomenclatura de Branches

```
feature/descricao-clara          # Novas funcionalidades
fix/descricao-do-erro            # Correção de bugs
refactor/descricao-melhoria      # Refatoração de código
docs/descricao-documentacao      # Atualizações de docs
```

### Mensagens de Commit

Use Conventional Commits:
- `feat:` Novas funcionalidades
- `fix:` Correção de bugs
- `refactor:` Refatoração
- `docs:` Documentação
- `style:` Formatação
- `test:` Testes

### Checklist Antes de Submeter

- [ ] Código segue o estilo do projeto
- [ ] Testes foram adicionados
- [ ] Documentação foi atualizada
- [ ] Sem console.log deixados
- [ ] TypeScript compila sem erros
- [ ] Testes passam
